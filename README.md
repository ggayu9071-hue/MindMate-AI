# 🧠 MindMate AI

An AI-powered Mental Wellness Platform built using **Flask**, **Google Gemini AI**, and **SQLite**.

MindMate AI provides a safe and supportive environment where users can chat with an AI assistant, track their daily moods, and maintain a personal journal to improve their mental well-being.

---

## ✨ Features

- 🔐 User Registration & Login
- 🤖 AI Mental Wellness Chatbot (Google Gemini)
- 😊 Daily Mood Tracker
- 📖 Personal Journal
- 📊 Dashboard
- 🔒 Secure Authentication using Flask-Login
- 💾 SQLite Database
- 🎨 Responsive User Interface

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/MindMate-AI.git
```

Go into the project

```bash
cd MindMate-AI
```

Create virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Install packages

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
SECRET_KEY=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
```

Run

```bash
python app.py
```
